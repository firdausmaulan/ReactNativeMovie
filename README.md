#react-native-movie

Tech Stack : 
- MVC
- Axios http client
- Async Storage
- React Navigation

Screen
- Splash Screen
- Login Screen
- Movie List Screen
- Movie Detail Screen

How to build
- start metro : npx react-native start
- i : run on IOS
- a : run on android
